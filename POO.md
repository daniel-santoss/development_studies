| Nome | O que é |
|------|--------|
| Herança | É um mecanismo que permite criar uma nova classe a partir de uma já existente reutilizando seus atributos e métodos. Exemplo: uma classe pai "VEÍCULO" possui o atributo motor e o método andar, e a classe filha "CARRO" herda essas características. |
| Polimorfismo | Permite que diferentes classes respondam de maneira diferente a um mesmo método. Exemplo: a classe "ANIMAL" possui o método fazerSom. As classes "CACHORRO" e "GATO" implementam esse método retornando "AU AU" e "MIAU". |
| Encapsulamento | É um conceito que protege os atributos e métodos de uma classe, permitindo controlar quem pode acessá-los. Exemplo: a classe "CONTA BANCÁRIA" possui o atributo saldo privado, acessado apenas por métodos públicos como depositar e sacar. |
| Construtores | São métodos especiais usados para iniciar um objeto já com valores definidos no momento em que ele é criado. Exemplo: a classe "PESSOA" recebe nome e idade diretamente no construtor. |
| Classe Abstrata | É uma classe que não pode ser instanciada diretamente e serve como um modelo para outras classes. Exemplo: a classe abstrata "FUNCIONÁRIO" define o método calcularSalario, implementado de forma específica por "GERENTE" e "VENDEDOR". |
| Interface | É um tipo especial que define apenas métodos (sem implementação), servindo como um contrato que as classes devem cumprir. Exemplo: a interface "AUTENTICÁVEL" obriga classes como "USUÁRIO" e "ADMIN" a implementarem o método autenticar. |
| Testes Unitários | São testes usados para verificar se partes pequenas e específicas do código funcionam corretamente de forma isolada. Exemplo: testar o método somar da classe "CALCULADORA" para garantir que 2 + 3 resulte em 5. |
| Escopo do Projeto | É a definição clara de tudo o que será feito e do que não será feito dentro de um projeto. Exemplo: em um aplicativo de delivery, o escopo inclui cadastro e pedidos, mas exclui pagamento online na primeira versão. |
| .NET | É uma plataforma da Microsoft usada para desenvolver aplicações web, desktop, serviços e jogos. As versões mais antigas funcionam apenas no Windows e oferecem diversas bibliotecas prontas. |
| .NET Core | É uma versão mais moderna, rápida e multiplataforma do .NET, compatível com Windows, Linux e macOS, sendo mais leve e ideal para aplicações modernas. |
